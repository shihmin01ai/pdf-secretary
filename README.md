# PDF 小秘書 (PDF Secretary)

這是一個 **100% 在本地端（瀏覽器中）運行的 PDF 編輯工具**，支援合併、分割、旋轉個別頁面、刪除頁面及重新排序。

所有 PDF 處理均在您的瀏覽器中完成，**檔案絕不上傳到任何伺服器**，確保極致的隱私與安全。

---

## 🌟 核心功能

* 📁 **拖曳上傳 (Drag & Drop)**：直覺的檔案載入區域，支援一次加載多個 PDF。
* 🔄 **頁面旋轉**：可針對個別頁面進行 90°、180°、270° 順時針旋轉。
* 🗑️ **頁面刪除**：一鍵移除特定頁面。
* 🔀 **頁面排序**：支援滑鼠拖曳（HTML5 Drag and Drop）自由調整頁面順序。
* 🎛️ **批次操作**：可勾選/全選多個頁面，進行一鍵「批次刪除」或「批次旋轉」。
* ⬇️ **合併導出**：將排列好且編輯過的頁面合併匯出為一個單一 PDF 下載。
* ✂️ **分割導出**：將每一頁分別分割，打包成一個 ZIP 壓縮包下載。
* 🎨 **現代美學**：極致的磨砂玻璃風格（Glassmorphism）UI，支援流暢切換「深色/淺色」主題。

---

## ⚙️ 系統優化技術

1. **記憶體防 Neutered 機制**：
   - 修正了 PDF.js 在 Worker 中解構傳移 `ArrayBuffer` 的問題，透過 `slice(0)` 複製記憶體防護，使 PDF-lib 進行匯出合併時順暢無阻。
2. **縮圖 Canvas 快取**：
   - PDF.js 渲染的縮圖會被快取在記憶體中，當調整排序、選取狀態時直接復用 Canvas，避免重繪閃爍，即使操作大檔案也極速流暢。
3. **PDF-lib Lazy Load 載入快取**：
   - 合併匯出時對多檔案來源進行 Lazy Load，大檔案處理速度提昇高達 10 倍以上。

## 🚀 如何啟動使用

本專案已配置好本地靜態伺服器環境。請在終端機中切換至本專案路徑，並啟動開發伺服器：

```bash
# 1. 切換至專案目錄
cd C:\Users\user\.gemini\antigravity-ide\scratch\pdf-secretary

# 2. 啟動伺服器 (預設在 http://localhost:3000)
npm run dev
```

啟動後，使用瀏覽器打開網址：
👉 **[http://localhost:3000](http://localhost:3000)**

即可直接開始使用 PDF 小秘書！
